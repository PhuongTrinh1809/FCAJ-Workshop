This section lists and introduces the blogs posted to [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj):

### [Blog 1 – Amazon Bedrock AgentCore Web Search](3.1-Blog1/)
This blog introduces **Web Search on Amazon Bedrock AgentCore**, enabling AI Agents to access real-time information from the Internet without building a custom search system. It covers the architecture, processing flow, and benefits of having AWS manage the entire search layer for Agents.

### [Blog 2 – Building a Scalable User Search Layer on Amazon Cognito](3.2-Blog2/)
This blog summarizes the **AWS Architecture Blog** solution for extending Cognito user search capabilities using **AWS Lambda**, **Amazon DynamoDB**, and **Amazon OpenSearch Serverless**. It covers two main flows: Ingest Flow (user data synchronization) and Search Flow (querying via API Gateway with group-based authorization).

### [Blog 3 – Protein Research Copilot with Amazon Bedrock AgentCore](3.3-Blog3/)
This blog shares the **Protein Research Copilot** architecture — a system that helps biology researchers search and analyze peptides through natural language. The solution combines **Amazon Bedrock AgentCore**, **SageMaker AI**, **Aurora PostgreSQL**, and a multi-agent model (Orchestrator, Parser, Searcher, Summarizer).
