1. Search `VPC` → đảm bảo region **ap-southeast-1**.
2. **Your VPCs** → **Create VPC**.
3. Chọn **VPC only**.
4. Name tag: `zerobug-vpc`.
5. IPv4 CIDR: **`10.0.0.0/16`**.
6. → **Create VPC**.
7. Copy **VPC ID** → bảng tham số.
